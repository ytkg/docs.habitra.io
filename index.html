HabiTra

Habit Tracker operated by API

Usage

User

Create User

$ curl -X POST https://api.habitra.io/v1/users -d '{"id":"takagi","password":"thisispassword"}'
{"message":"Success."}

Update User

$ curl -X PUT --user takagi:thisispassword https://api.habitra.io/v1/users/takagi -d '{"password":"thisisnewpassword"}'
{"message":"Success."}

Delete User

$ curl -X DELETE --user takagi:thisisnewpassword https://api.habitra.io/v1/users/takagi
{"message":"Success."}

Habit

Create Habit

$ curl -X POST --user takagi:thisispassword https://api.habitra.io/v1/users/takagi/habits -d '{"name":"Take a Yakult"}'
{"message":"Success."}

List Habit

$ curl -X GET --user takagi:thisispassword https://api.habitra.io/v1/users/takagi/habits
{"habits":[{"id":"01G9QNY2FRQRFQGBK6Q0N6EE7B","name":"Take a Yakult"}]}

Update Habit

$ curl -X PUT --user takagi:thisispassword https://api.habitra.io/v1/users/takagi/habits/01G9QNY2FRQRFQGBK6Q0N6EE7B -d '{"name":"Take a sip of Yakult"}'
{"message":"Success."}

Delete Habit

$ curl -X DELETE --user takagi:thisispassword https://api.habitra.io/v1/users/takagi/habits/01G9QNY2FRQRFQGBK6Q0N6EE7B
{"message":"Success."}

Track

Create Track

$ curl -s -X POST --user takagi:thisispassword https://api.habitra.io/v1/users/takagi/habits/01G9QNY2FRQRFQGBK6Q0N6EE7B/tracks
{"message":"Success."}

or

$ curl -s -X POST --user takagi:thisispassword https://api.habitra.io/v1/users/takagi/habits/01G9QNY2FRQRFQGBK6Q0N6EE7B/tracks -d '{"date":"2022-01-01"}'
{"message":"Success."}

List Track

$ curl -s -X GET --user takagi:thisispassword https://api.habitra.io/v1/users/takagi/habits/01G9QNY2FRQRFQGBK6Q0N6EE7B/tracks
{"tracks":[{"date":"2022-07-22"}]}

Delete Track

$ curl -s -X DELETE --user takagi:thisispassword https://api.habitra.io/v1/users/takagi/habits/01G9QNY2FRQRFQGBK6Q0N6EE7B/tracks/2022-07-22
{"message":"Success."}
